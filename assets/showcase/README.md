## 维护约定

- 新增展示素材时优先放在本目录，并在本文件登记用途。
- README 中引用本目录文件时使用相对路径，带空格的文件名用 `%20` 编码。
- 当前目录会随项目一起上传到 GitHub，不会被 `.gitignore` 忽略。
